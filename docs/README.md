# BunPress Docs Website

This folder contains the Docusaurus-powered BunPress documentation site.

## Local Development

```bash
bun run docs:dev
```

Or directly from this folder:

```bash
npm run start
```

## Build

```bash
bun run build:docs
```

Or directly from this folder:

```bash
npm run build
```

The generated static site is written to `build/`.
